Instructions:
1. Open the specified map above in Unreal Engine
2. Press Alt+P, or enter play mode in the level editor
3. Click anywhere on the viewport window so the game can receive user inputs
4. WASD to move around, mouse inputs to look around, QE to either go up or down
5. Press 1 and 2 to change what's displayed in the HUD. 1 for the previous widgets and 2 for the next widgets
